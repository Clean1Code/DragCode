# DragCode

DragCode is a visual programming language inspired by block-based coding environments like Scratch, rebuilt using modern web technologies.

Built with:
- React
- Vite
- Zustand
- TailwindCSS

## Features

- Drag and drop programming blocks
- Visual scripting interface
- State management using Zustand
- Fast development with Vite
- Responsive UI with TailwindCSS
- Modular and extensible architecture

## Tech Stack

| Technology | Purpose |
|------------|---------|
| React | Frontend UI |
| Vite | Build tool |
| Zustand | State management |
| TailwindCSS | Styling |
| React Router DOM | Routing |

---

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/DragCode.git
```

Navigate into the project:

```bash
cd DragCode
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

---

## Build for Production

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
